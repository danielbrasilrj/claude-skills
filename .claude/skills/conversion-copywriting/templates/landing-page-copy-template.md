# Landing Page Copy Template

Use this template for any conversion-focused landing page. Fill in all sections with specific, benefit-driven copy. Apply AIDA framework throughout.

---

**Product Name:** [Your product]

## Section 1: Hero (Above the Fold)

### Purpose
Capture attention in 3 seconds. Communicate the core benefit instantly.

### Copy Structure

#### Headline (Attention)
**Requirements:**
- Lead with the strongest benefit (not product name)
- Under 70 characters
- Specific number or outcome if possible
- Pass the "so what?" test

**English:**
```
[Benefit-driven headline with specific outcome]
```

**pt-BR:**
```
[Transcreated headline — not literal translation]
```

**Examples:**
- "Book Appointments in 30 Seconds, Not 30 Minutes"
- "Agende Consultas em 30 Segundos, Não em 30 Minutos"

---

#### Subheadline (Interest)
**Requirements:**
- Expand on the headline benefit
- Address "how" or "why this matters"
- 1-2 short sentences

**English:**
```
[Subheadline explaining the mechanism or value]
```

**pt-BR:**
```
[Subheadline adaptation — account for +25% text expansion]
```

**Examples:**
- "Our AI scheduler syncs calendars, sends reminders, and reschedules conflicts automatically."
- "Nosso assistente inteligente sincroniza agendas, envia lembretes e reagenda conflitos automaticamente."

---

#### Primary CTA (Action)
**Requirements:**
- Action verb + value proposition
- Friction reducer ("Free," "No credit card," etc.)
- Button text under 25 characters
- Single CTA above the fold

**English:**
```
Button: [Action + Value]
Microcopy: [Friction reducer]
```

**pt-BR:**
```
Button: [Action + Value in pt-BR]
Microcopy: [Friction reducer adapted]
```

**Examples:**
- Button: "Start Free Trial"
- Microcopy: "No credit card required • Cancel anytime"
- Button: "Começar Grátis"
- Microcopy: "Sem cartão de crédito • Cancele quando quiser"

---

#### Hero Image/Visual
**Requirements:**
- Show the product in use (not just a logo)
- Include a person if possible (builds trust)
- Demonstrate the outcome, not the interface

**Description:**

## Copywriting Checklist

Before publishing, verify:

**Hero:**
- [ ] Headline leads with benefit (not product name)
- [ ] CTA is action-driven and friction-free
- [ ] Hero visual shows product in use

**Body:**
- [ ] Problem section agitates pain points
- [ ] Benefits are outcome-focused (not feature lists)
- [ ] Social proof includes specific results
- [ ] Features have clear descriptions

**Localization:**
- [ ] pt-BR transcreated (not literally translated)
- [ ] Text expansion accounted for (+20-30%)
- [ ] Currency and dates formatted correctly
- [ ] "Você" tone throughout

**Conversion:**
- [ ] Single clear CTA per section
- [ ] Urgency/scarcity present (but not fake)
- [ ] Objections addressed in FAQ
- [ ] Mobile-friendly (short paragraphs, scannable)
