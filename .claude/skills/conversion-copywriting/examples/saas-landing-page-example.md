## Features

### Feature List

| Feature (EN) | Description (EN) | Feature (pt-BR) | Description (pt-BR) |

| Smart Reminders | Send automatic SMS and email reminders 24 hours before appointments | Lembretes Inteligentes | Envie lembretes automáticos por SMS e email 24 horas antes das consultas |
