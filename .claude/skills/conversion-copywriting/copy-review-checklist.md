# Copy Review Checklist

## Before Publishing Any Copy:

**Clarity:**

- [ ] Passes "5-year-old test" — simple, jargon-free
- [ ] One idea per sentence
- [ ] Active voice, second person ("you")

**Conversion:**

- [ ] Leads with strongest benefit
- [ ] Includes specific numbers/proof
- [ ] Single clear CTA
- [ ] Reduces friction (free, instant, risk-free)

**Localization (pt-BR):**

- [ ] Transcreated, not translated
- [ ] Text expansion accounted for (+20-30%)
- [ ] Numbers, dates, currency formatted correctly
- [ ] Uses "voce" tone, not formal

**Testing:**

- [ ] Hypothesis documented for A/B test
- [ ] Variants test one variable only
- [ ] Minimum sample size planned (100+ conversions/variant)
