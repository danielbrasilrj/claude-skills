# Brazilian Portuguese Copy Adaptation

## Transcreation vs Translation

**Translation** (literal):

- "Sign up now" → "Cadastre-se agora"

**Transcreation** (adapted):

- "Sign up now" → "Comece gratis" ("Start for free" — more compelling in pt-BR context)

## Cultural Adaptation Rules

### 1. Formality Level

- **Use "voce"** (informal, friendly) — not "o senhor/a senhora" (formal)
- Exception: Legal, financial, or government contexts use formal tone

### 2. Emotional Triggers

Brazilian audiences respond strongly to:

- **Community/belonging** — "Junte-se a milhares de pessoas" (Join thousands)
- **Status/exclusivity** — "Acesso VIP" (VIP access)
- **Family/care** — "Cuide de quem voce ama" (Care for those you love)

### 3. Call-to-Action Patterns

| English     | Direct Translation | Better pt-BR Adaptation                     |
| ----------- | ------------------ | ------------------------------------------- |
| Get started | Comece             | Comecar agora / Quero comecar               |
| Try free    | Experimente gratis | Testar gratis / Experimente sem compromisso |
| Learn more  | Saiba mais         | Ver como funciona / Conhecer melhor         |
| Buy now     | Compre agora       | Quero comprar / Garantir meu desconto       |
| Sign up     | Cadastre-se        | Criar conta gratis / Comecar gratis         |

**Pattern:** pt-BR prefers "I want to" (Quero) over imperative commands

### 4. Text Expansion Guidelines

| English Length | Expansion Factor | Example                                                      |
| -------------- | ---------------- | ------------------------------------------------------------ |
| 1-10 chars     | +30%             | "Save" (4) → "Salvar" (6)                                    |
| 11-20 chars    | +25%             | "Get started" (11) → "Comecar agora" (13)                    |
| 21-40 chars    | +20%             | "Book your free trial" (20) → "Comece seu teste gratis" (25) |
| 40+ chars      | +15%             | Full paragraphs                                              |

**Design implications:**

- Never use fixed-width containers for buttons
- Allow 2 lines for single-line labels
- Test UI with longest pt-BR strings

### 5. Number and Currency Formatting

| Type        | English    | pt-BR           |
| ----------- | ---------- | --------------- |
| Decimal     | 1,234.56   | 1.234,56        |
| Currency    | $49.99     | R$ 49,99        |
| Dates       | 03/15/2025 | 15/03/2025      |
| Time        | 3:45 PM    | 15h45           |
| Percentages | 25% off    | 25% de desconto |

### 6. Urgency and Scarcity

**English:** "Only 5 spots left"
**pt-BR:** "Restam apenas 5 vagas" OR "Ultimas vagas disponiveis"

**English:** "Offer ends tonight"
**pt-BR:** "Oferta termina hoje a meia-noite" OR "Ultima chance — termina hoje"

Brazilian audiences respond well to:

- Time-based urgency (deadlines)
- Quantity scarcity (limited spots)
- Social proof urgency ("100 pessoas compraram hoje")

### 7. Social Proof Adaptation

**English:** "Trusted by 10,000+ customers"
**pt-BR:** "Mais de 10 mil clientes confiam" OR "Escolhido por 10 mil pessoas"

**Pattern:** pt-BR uses "mais de" (more than) instead of "10,000+"

### 8. Common Pitfalls to Avoid

| Avoid                     | Use Instead             | Why                   |
| ------------------------- | ----------------------- | --------------------- |
| Literal English structure | Natural pt-BR flow      | Sounds robotic        |
| "Gratis gratis gratis"    | "Comece sem pagar nada" | Overuse reduces trust |
| Overly formal tone        | Conversational "voce"   | Feels cold/distant    |
| American idioms           | Local expressions       | Lost in translation   |
| Generic benefits          | Specific outcomes       | Low engagement        |

### 9. Power Words in pt-BR

| Category    | Words                                           |
| ----------- | ----------------------------------------------- |
| Urgency     | Agora, hoje, ultima chance, restam apenas       |
| Value       | Gratis, sem custo, bonus, desconto exclusivo    |
| Ease        | Facil, simples, rapido, em minutos              |
| Proof       | Comprovado, testado, garantido, aprovado        |
| Exclusivity | Exclusivo, VIP, acesso especial, membros apenas |

### 10. Email Subject Line Adaptations

**English:** "Your free trial ends tomorrow"
**pt-BR:** "Seu teste gratis termina amanha"

**English:** "50% off — today only"
**pt-BR:** "50% OFF — so hoje!"

**Notes:**

- pt-BR uses more emojis in subject lines (higher open rates)
- Prefers "voce" (your) for personalization
- "OFF" is commonly used untranslated for discounts
