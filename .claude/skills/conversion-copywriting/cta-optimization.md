# CTA Optimization

## CTA Anatomy

Every CTA needs:

1. **Action verb** — Start with "Get," "Start," "Claim," "Join," "Try"
2. **Value proposition** — What they get (not what they do)
3. **Friction reducer** — "Free," "No credit card," "Cancel anytime"

## High-Converting CTA Patterns

| Pattern            | Example                     | When to Use                 |
| ------------------ | --------------------------- | --------------------------- |
| **Benefit-driven** | "Get My Free Guide"         | Lead magnets, downloads     |
| **Time-bound**     | "Start Your 14-Day Trial"   | Free trials, limited offers |
| **Social proof**   | "Join 10,000+ Users"        | Community-driven products   |
| **Urgency**        | "Claim Your Spot (5 Left)"  | Scarcity-based offers       |
| **Curiosity**      | "See How It Works"          | Demo videos, product tours  |
| **Zero-risk**      | "Try Free — No Credit Card" | High-friction signups       |

## CTA Testing Checklist

- [ ] Uses action verb (not "Submit" or "Click here")
- [ ] Specifies value ("Get," not "Learn more")
- [ ] Reduces friction (free, instant, no commitment)
- [ ] Stands out visually (color contrast, whitespace)
- [ ] Single CTA per section (avoid choice paralysis)
- [ ] Mobile thumb-friendly (min 44x44px tap target)
