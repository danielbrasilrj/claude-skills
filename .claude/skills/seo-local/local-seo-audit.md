# Local SEO Audit Workflow

## 1. GBP Audit (30 min)

- [ ] All fields 100% complete
- [ ] Primary + 5 secondary categories selected
- [ ] 10+ geotagged photos uploaded
- [ ] 5+ services added with descriptions
- [ ] Business hours accurate (including holidays)
- [ ] Posts published in last 7 days
- [ ] Q&A section has 5+ pre-answered questions
- [ ] Respond to all reviews (last 30 days)

---

## 2. NAP Audit (60 min)

- [ ] NAP matches exactly across GBP, website, Apple Maps, Bing
- [ ] Check top 20 Google results for `"[business name]" "[city]"` for inconsistencies
- [ ] Verify NAP on Yelp, Facebook, LinkedIn
- [ ] Fix any abbreviation mismatches (St vs Street)

---

## 3. Citation Audit (90 min)

- [ ] Claimed profiles on Tier 1 directories (Google, Apple, Bing, Yelp, Facebook)
- [ ] Claimed profiles on data aggregators (Neustar, Foursquare, Acxiom)
- [ ] Claimed 10+ industry-specific directories
- [ ] Run Moz Local scan for citation gaps
- [ ] Fix or remove duplicate listings

---

## 4. On-Page SEO Audit (60 min)

- [ ] LocalBusiness schema on every location page
- [ ] FAQ schema on main service pages
- [ ] Hreflang tags if multilingual
- [ ] Title tags include geo-modifier ("[Service] in [City]")
- [ ] H1 includes location ("[City] [Service]")
- [ ] Embed Google Map on contact page
- [ ] Footer NAP present on every page

---

## 5. Review Audit (30 min)

- [ ] 4.0+ star average
- [ ] 5+ reviews in last 30 days
- [ ] Respond to 100% of reviews (positive + negative)
- [ ] Set up automated review request system
- [ ] Monitor review platforms weekly

---

## Advanced Local SEO Topics

### Local Link Building

**Goal:** Get backlinks from local websites to boost geographic relevance.

**Strategies:**

1. **Local sponsorships:** Sponsor Little League team, charity event -> get link from event page
2. **Local news:** Pitch story to local newspaper/blog -> earn editorial link
3. **Chamber of Commerce:** Join local business association -> directory link
4. **Local suppliers:** Ask vendors to list you as customer -> link from their site
5. **Local partnerships:** Cross-promote with complementary businesses -> reciprocal links

**Quality > quantity:** 1 link from local newspaper > 10 links from random directories.

---

### Google Posts SEO

Google Posts don't directly affect rankings, but increase CTR (click-through rate), which IS a ranking signal.

**Post types:**

- **Weekly update:** "New service available", "Blog post published"
- **Monthly offer:** "10% off this month" (with promo code)
- **Quarterly event:** "Open house", "Grand reopening"

**Best practices:**

- 100-300 words (long enough to be useful, short enough to read quickly)
- Add CTA button (Book Now, Learn More, Call Now)
- Include photo (1200x900px recommended)
- Post Monday-Friday 9am-12pm (highest engagement window)

---

### Voice Search Optimization

**Voice searches are 3x more likely to be local** ("near me" queries).

**Optimization:**

- Target question keywords ("where is", "how do I", "what's the best")
- Answer in FAQ format (voice assistants pull from FAQ schema)
- Use conversational language (how people speak, not type)
- Optimize for featured snippets (position 0 = voice search answer)

**Example:**

- **Typed query:** "dentist austin tx"
- **Voice query:** "Where's the best dentist near me?"

**Optimization:** Add FAQ "What's the best dentist in Austin?" with answer highlighting your services.

### Local Inventory Ads (Retail)

If you sell products in-store, enable **Local Inventory Ads** (LIA).

**How it works:**

1. Upload product inventory feed to Google Merchant Center
2. Google shows "In stock nearby" for local searches
3. Users click -> see store location + directions

**Requirements:**

- Google Merchant Center account
- Product feed with inventory levels
- GBP linked to Merchant Center

**Use case:** Retail stores with physical inventory (electronics, apparel, home goods).
