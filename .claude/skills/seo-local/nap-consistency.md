# NAP Consistency (Critical)

**NAP = Name, Address, Phone**

Google's algorithm matches NAP across the web to verify your business is legitimate. Inconsistencies = trust penalty = lower rankings.

## NAP Audit Checklist

Check these locations for 100% identical NAP:

**Primary:**

- [ ] Google Business Profile
- [ ] Website footer
- [ ] Website contact page
- [ ] Apple Maps (https://mapsconnect.apple.com)
- [ ] Bing Places (https://www.bingplaces.com)

**Directories:**

- [ ] Yelp
- [ ] Facebook Business Page
- [ ] LinkedIn Company Page
- [ ] Yellow Pages
- [ ] Industry-specific directories (Avvo for lawyers, Healthgrades for doctors, etc.)

**Social Media:**

- [ ] Instagram bio
- [ ] Twitter bio
- [ ] YouTube About

## Common NAP Inconsistencies

| Issue                | Example                                               | Fix                                                |
| -------------------- | ----------------------------------------------------- | -------------------------------------------------- |
| **Abbreviations**    | "123 Main St" vs "123 Main Street"                    | Standardize (use full "Street" everywhere)         |
| **Suite numbers**    | "123 Main St" vs "123 Main St Suite 200"              | Include suite consistently                         |
| **Phone formatting** | "(512) 555-0100" vs "512-555-0100"                    | Standardize format (either is fine, just pick one) |
| **Business name**    | "Green Valley Dental" vs "Green Valley Dental Clinic" | Use exact legal name everywhere                    |

**Tools to find NAP inconsistencies:**

- **Moz Local** (paid, $99/year)
- **Yext** (paid, starts $199/year)
- **Manual Google search:** Search `"your business name" "your city"` and check all results
