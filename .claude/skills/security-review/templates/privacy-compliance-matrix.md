# Privacy Compliance Matrix: LGPD vs GDPR vs CCPA

## Regulation Overview

| Aspect | LGPD (Brazil) | GDPR (EU/EEA) | CCPA/CPRA (California) |
|--------|---------------|---------------|------------------------|
| **Effective** | Sep 2020 (penalties Aug 2021) | May 2018 | Jan 2020 (CPRA: Jan 2023) |
| **Authority** | ANPD | National DPAs | California AG / CPPA |
| **Scope** | Processing of data in Brazil or of data subjects in Brazil | Processing of EU/EEA residents' data | Businesses meeting CA thresholds |

## Applicability Thresholds

| Criterion | LGPD | GDPR | CCPA/CPRA |
|-----------|------|------|-----------|
| Revenue threshold | None (applies to all) | None (applies to all) | >$25M annual revenue |
| Data volume | None | None | 100k+ consumers/households |
| Revenue from data | None | None | 50%+ from selling/sharing PI |
| Geographic trigger | Data subjects in Brazil OR processing in Brazil | Data subjects in EU/EEA | Doing business in California |

|-------------|---------------|---------------|-----------|

| Right | LGPD | GDPR | CCPA/CPRA |
|-------|------|------|-----------|
| Right to know / access | Yes (Art. 18) | Yes (Art. 15) | Yes |
| Right to correction | Yes | Yes (Art. 16) | Yes (CPRA) |
| Right to deletion | Yes | Yes (Art. 17) | Yes |
| Right to portability | Yes | Yes (Art. 20) | Yes (CPRA enhanced) |
| Right to object | Yes | Yes (Art. 21) | N/A |
| Right to restrict processing | Yes (anonymization/blocking) | Yes (Art. 18) | Yes (limit sensitive PI) |
| Right to opt-out of sale | N/A | N/A | Yes |
| Right to opt-out of sharing | N/A | N/A | Yes (CPRA) |
| Right to non-discrimination | Yes | Yes | Yes |
| Consent revocation | Yes | Yes | N/A (opt-out model) |
| Right to information about sharing | Yes | Yes | Yes |

## Response Timelines

| Requirement | LGPD | GDPR | CCPA/CPRA |
|-------------|------|------|-----------|
| DSAR response | **15 days** | 30 days (extendable +60) | 45 days (extendable +45) |
| Breach notification (authority) | Reasonable time | **72 hours** | N/A (AG notification required for 500+ CA residents) |
| Breach notification (subjects) | Reasonable time | Without undue delay | Expeditious |
| Privacy policy update | Not specified | As needed | At least every 12 months |

## Penalties

| Penalty Type | LGPD | GDPR | CCPA/CPRA |
|-------------|------|------|-----------|
| Max fine | **R$50M** per violation or 2% of revenue | **EUR 20M** or 4% of global turnover | $2,500 per violation |
| Intentional violation | Same | Same (higher tier) | **$7,500** per violation |
| Daily fine | Yes | Yes (some DPAs) | N/A |
| Data blocking/deletion | Yes | Yes | N/A |
| Private right of action | Limited | Yes (Art. 82) | Yes (data breaches: $100-$750/consumer) |
| Warning | Yes (first tier) | Yes | N/A |
| Public disclosure | Yes | Yes | N/A |

## Implementation Checklist

### For All Three Regulations
- [ ] Privacy policy published and accessible
- [ ] Data inventory / mapping completed
- [ ] Data subject/consumer request handling process
- [ ] Breach detection and notification process
- [ ] Vendor/third-party data processing agreements
- [ ] Employee training on data protection
- [ ] Data retention schedule defined
- [ ] Security measures documented

### LGPD-Specific
- [ ] Encarregado (DPO) appointed
- [ ] Legal basis documented for each processing activity
- [ ] RIPD (Data Processing Impact Assessment) completed
- [ ] 15-day DSAR response process tested
- [ ] ANPD registration completed (when required)

### GDPR-Specific
- [ ] DPO appointed (if required by Art. 37)
- [ ] Lawful basis documented (Art. 6)
- [ ] DPIA completed for high-risk processing (Art. 35)
- [ ] Records of processing activities (Art. 30)
- [ ] 72-hour breach notification process tested
- [ ] Data transfer impact assessments (Schrems II)

### CCPA/CPRA-Specific
- [ ] "Do Not Sell or Share My Personal Information" link on website
- [ ] Two or more methods for consumer requests
- [ ] 12-month lookback capability for data requests
- [ ] Service provider and contractor agreements updated
- [ ] Privacy policy updated within last 12 months
- [ ] Annual metrics published (if 10M+ consumer requests)
